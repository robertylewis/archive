Here is an NNG proof:

  split,
  all_goals { intro h },
  all_goals { repeat { cases h with h h } },
  all_goals {
    try { left; split; assumption },
    try { right; split; assumption } },
  all_goals { split },
  all_goals { try { assumption } },
  try { left; assumption },
  try { right; assumption},
